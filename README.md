# LinksValidator


REQUIREMENTS
------------

- Node.js 10.15

SET UP THE DEV ENVIROMENT
------------

Create docker nodejs:

```
$ docker build -t linkValidator .
```

FIRST TIME RUNNING 
------------------

```
$ docker-compose up -d

```
INSTALL DEPENDECECIES:
---------------------

Run inside the container the follow commad
- npm install
